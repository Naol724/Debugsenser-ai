# PWA Icons

This folder contains the Progressive Web App icons for DebugSense AI.

## Required Icons

You need to create the following icon files:

1. **icon-192x192.png** (192x192 pixels)
   - Used for Android devices and smaller screens
   - Should be a square with transparent background
   - Recommended: Blue/gradient theme with "DS" or bug icon

2. **icon-512x512.png** (512x512 pixels)
   - Used for splash screens and larger displays
   - Should be a square with transparent background
   - Same design as 192x192 but higher resolution

## Design Guidelines

- **Colors**: Use the app theme colors (#0f172a, #6366f1, #8b5cf6)
- **Style**: Modern, clean, professional
- **Content**: Consider using:
  - "DS" monogram
  - Bug/debug icon
  - Code brackets with bug
  - Simple geometric design

## How to Create Icons

### Option 1: Online Tools
1. Go to [favicon.io](https://favicon.io/)
2. Upload your logo or create one online
3. Download the 192x192 and 512x512 versions
4. Place them in this folder

### Option 2: Design Software
1. Use Canva, Figma, or Adobe Illustrator
2. Create 192x192 and 512x512 square designs
3. Export as PNG with transparency
4. Place them in this folder

### Option 3: Quick Placeholder (for testing)
You can temporarily use any square PNG images with these dimensions to test the PWA functionality.

## File Names
- `icon-192x192.png`
- `icon-512x512.png`

Make sure the file names match exactly what's specified in the manifest.json file.
